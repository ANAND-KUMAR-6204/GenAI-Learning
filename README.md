---

### 02. GAN on Fashion-MNIST

Implementation of a Generative Adversarial Network (GAN) trained on the Fashion-MNIST dataset.

The model consists of a Generator that learns to create synthetic fashion images from random noise and a Discriminator that learns to distinguish generated samples from real images. Through adversarial training, both networks continuously improve, allowing the Generator to generate increasingly realistic outputs.

**Concepts Explored**

* Generative Adversarial Networks (GANs)
* Adversarial Learning
* Generator–Discriminator Architectures
* Image Generation
* Custom Training Loops using `tf.GradientTape`
* TensorFlow Model Development

---

### 03. Variational Autoencoder (VAE)

Implementation of a Variational Autoencoder (VAE) using TensorFlow to explore latent space learning and probabilistic generative modeling.

The model learns compressed latent representations of data and uses them to reconstruct existing samples as well as generate new ones. This implementation provides practical insight into how latent spaces capture meaningful information and support data generation.

**Concepts Explored**

* Variational Autoencoders (VAEs)
* Encoder–Decoder Architectures
* Latent Space Representation
* Probabilistic Generative Modeling
* Image Reconstruction
* Representation Learning
